# Maldad Políglota

> Un programa verdaderamente **políglota**, capaz de compilar y ejecutarse en múltiples lenguajes:  
> **C**, **C++**, **Objective-C**, **Whitespace** y **PHP**.  

---

## Descripción

Este proyecto demuestra cómo un mismo archivo fuente puede ser válido en varios lenguajes de programación distintos.  
El archivo principal (`maldad`) ha sido cuidadosamente diseñado para que compile y ejecute sin modificaciones en los lenguajes listados abajo.

---

## Lenguajes soportados

- **C**  
- **C++**  
- **Objective-C**  
- **Whitespace**  
- **PHP**

---

## Archivo principal
    maldad

----------------------------------------
## Ejecución en los diferentes lenguajes:
    -C:             `$ gcc maldad -o maldad
$ ./maldad`

    -C++:            `$ g++ maldad -o maldad
./maldad`

    -Objective C:    `Copiar código en Ideone.com y ejecutar usando Objective C` 

    -PHP:            `php maldad`

    -WhiteSpace:     `Copiar código en https://vii5ard.github.io/whitespace/ y preisonar "Run"`
